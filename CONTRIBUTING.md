Here is the prettier style to follow when contributing:

"trailingComma": "none",
  "semi": true,
  "singleQuote": false,
  "bracketSameLine": true,
  "htmlWhitespaceSensitivity": "ignore",
  "proseWrap": "always",
  "printWidth": 64

  We also use the default tabWidth of 2.

  Make sure to install the Prettier and ESLint extensions!
